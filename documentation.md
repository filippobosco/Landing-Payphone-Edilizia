## NOTE TECNICHE PER L'IMPLEMENTAZIONE

### Stack Tecnologico
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS 4.1+
- **Animations:** Framer Motion 12
- **Icons:** Heroicons
- **Backgrounds:** Pattern edilizi animati custom (elementi geometrici, blueprints stilizzati)

### Framer Motion - Animazioni Chiave
1. **Hero:** Foto statica con overlay gradient + fade-in del contenuto
2. **Feature Cards:** Stagger children con fade-in-up
3. **Stats:** Counter animation con easing quando entra in viewport
4. **Steps:** Sequential reveal con scale effect
5. **Codice Promo:** Pulse + glow infinito
6. **Urgenze:** Grid stagger con hover scale + glow colorato per categoria
7. **CTA Finale:** Parallax background + button pulse

### Tailwind - Classi Custom Suggerite
- Gradient Payphone Edile: `bg-gradient-to-br from-[#F76a00] via-[#00b7ff] to-[#a03478]`
- Hero Overlay: `bg-gradient-to-r from-slate-900/75 via-slate-900/60 to-slate-900/50`
- Shadow system: `shadow-construction-soft` e `shadow-construction-strong`
- Border gradient per codice promo box con tonalità edili
- Custom spacing scale per sezioni (py-20, py-24, py-32)

### Heroicons - Set Principale
- `wrench-screwdriver` (interventi generici)
- `home-modern` (strutturale)
- `bolt` (urgenza/elettrico)
- `shield-check` (certificazione)
- `currency-euro` (trasparenza)
- `device-phone-mobile`
- `magnifying-glass`
- `phone`
- `gift`
- `exclamation-triangle` (emergenze)
- `fire` (caldaia)
- `document-text` (pratiche)

### Color Palette Edile
- Primary: #F76a00 (arancione costruzione)
- Secondary: #00b7ff (blu professionale)
- Accent: #a03478 (viola/magenta)
- Background: #FAFBFC
- Text: #1A202C

### Hero Background Image
- Foto professionale di equipe edile (4-5 professionisti in abbigliamento tecnico/professionale)
- Ambiente: cantiere moderno luminoso o ufficio tecnico con progetti
- Overlay gradient applicato via CSS: `from-slate-900/75 to-slate-900/50`
- Immagine ottimizzata per web (WebP, lazy load)
- Aspect ratio 16:9, min resolution 1920x1080

### Responsive Breakpoints
- Mobile-first approach
- Hero: foto full-width con overlay più scuro su mobile per leggibilità
- Feature cards: 1 col mobile, 3 col desktop
- Stats: 2 col mobile, 4 col desktop
- Steps: verticale mobile, orizzontale desktop con connettori
- Urgenze: 1 col mobile, 2 col tablet, 3 col desktop

### Performance Optimizations
- Hero image: WebP format con fallback JPEG
- Lazy loading per immagini sotto la fold
- Intersection Observer per animazioni viewport-triggered
- Preload degli store badges
- Ottimizzazione pattern animati (CSS transform invece di filter)

### Accessibilità
- Contrasti WCAG AA per tutti i testi
- Focus states chiari per navigazione da tastiera
- Aria-labels descrittivi per urgenze e CTA
- Alt text descrittivi per immagini hero
